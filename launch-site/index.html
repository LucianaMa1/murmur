current features

A tiny app with the controls you actually need.

Fn
Push-to-talk dictation

Hold Fn, speak, release. Murmur records and transcribes with WhisperKit locally on your Mac.

⌃
AI rewrite/polish

Hold Fn + Control to transcribe, then rewrite with your saved prompt. Only text goes to OpenAI, not audio.

☝
Floating controls

Use on-screen Transcribe and Rewrite buttons when macOS catches the Fn key.

⌘
Auto-paste + clipboard restoration

Paste directly into the active text field, then restore whatever was already on your clipboard.

Aa
Vocabulary support

Add names, jargon, commands, and correction pairs in plain-text vocabulary files.

bug

See recording, transcription, permission, paste, and API logs when something fails.

troubleshooting

When macOS gets picky, check these first.

Permissions matter

  • Microphone is required for recording.
  • Accessibility is required for auto-paste.
  • Input Monitoring is required for hotkeys.
  • If permissions act weird, remove and re-add Murmur in Accessibility and Input Monitoring.
  • After permission changes, quit and reopen Murmur.

If hotkeys or paste fail

  • If macOS intercepts Fn, use the floating Transcribe/Rewrite buttons.
  • Raw Fn transcription works locally.
  • Fn + Control rewrite needs internet and an OpenAI key.
  • English only for now.
  • Use the Debug Panel to check recording, transcription, paste, permissions, and API failures.
