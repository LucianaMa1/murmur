# Murmur

Murmur is a macOS menu-bar dictation app.

## Usage

- Hold F5 → raw transcription
- Hold F6 → polished (LLM)

## Features

- Menu bar app
- Hold-to-record
- Auto paste
- Vocabulary learning

## Requirements

- macOS 14+
- Xcode
- OpenAI API key (for F6)

## Status

Prototype. Needs Xcode project to run.
